# 06 MJ World History (2109010) - Fall Semester

GitHub Pages host for Optima Academy Online's 6th Grade World History (Dawn of Civilization to Roman Empire) Fall Semester course.

This repo hosts GitHub-deployed content embedded into Canvas course 104 via iframe:
- Interactive lesson pages (per `optima-lesson-format`)
- Interactive widgets (per `canvas-interactive-widgets`)
- Embedded timeline/map visuals

Structure:
```
week-01/ ... week-18/    per-week lesson pages, primary source pages, widgets
review-final/            Fall Semester review lesson
appendices/              supplemental standards assessments, glossary
```

Graded items (quizzes, assignments, discussions, module wrappers) are NOT here - they live natively in Canvas per `optima-canvas-assignments`.
